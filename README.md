🚦 Intelligent Traffic Signal Control System
📌 Overview

This project presents an Intelligent Traffic Signal Control System that uses Reinforcement Learning techniques such as Q-Learning and Deep Q-Networks (DQN) to reduce traffic congestion during peak hours. Traditional traffic signals work on fixed time intervals, which often causes unnecessary waiting and traffic buildup.

The proposed system dynamically adjusts traffic signal timings based on real-time vehicle density detected using sensors. By learning from traffic patterns, the system optimizes signal switching to reduce waiting time and improve traffic flow.

🎯 Objectives

Develop an adaptive traffic signal system using Reinforcement Learning.

Reduce traffic congestion and vehicle waiting time.

Dynamically adjust signal timings based on traffic density.

Simulate an intelligent traffic management system using IoT devices.

Provide a foundation for smart city traffic management systems.

⚙️ System Architecture

Sensors detect vehicle density on each road.

Sensor data is sent to the reinforcement learning agent.

The agent calculates the optimal traffic signal timing.

Traffic lights change dynamically according to traffic conditions.

The system continuously learns and improves traffic control decisions.
